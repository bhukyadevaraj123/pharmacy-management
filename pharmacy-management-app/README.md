
# ABC Pharmacy Assignment

## Features
- View medicines
- Add medicines
- Search medicines
- Red highlight for expiry < 30 days
- Yellow highlight for quantity < 10
- JSON file storage

---

## Backend Run

cd backend

dotnet restore
dotnet run

---

## Frontend Run

cd frontend

npm install
npm run dev

---

## API Endpoint

http://localhost:5000/api/medicines
